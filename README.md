# DUOS Contabilidade - Inteligência Contábil e Diagnóstico Fiscal

Este é o site institucional de alta performance da **DUOS Contabilidade**, um escritório de assessoria empresarial exclusiva, contabilidade estratégica e BPO financeiro estruturado no **Manhattan Office** em Santos, SP. 

O site foi desenvolvido com foco em conversão imediata de leads corporativos, mesclando a sofisticação de uma marca corporativa premium com a agilidade de ferramentas digitais modernas de conversão e simulação tributária.

---

## 🚀 Sobre o Site

O portal da DUOS Contabilidade apresenta o modelo de **Atendimento Contábil Híbrido** (Suporte digital veloz via WhatsApp unida à solidez de atendimento presencial de diretoria). Ele foi projetado para convencer empreendedores, comerciantes e prestadores de serviços de que é fácil e altamente vantajoso migrar de contabilidade para otimizar seus tributos.

### Principais Diferenciais e Funcionalidades:

1. **Aparência e Design Core Slate**:
   - Tema visual escuro (slate/indigo/blue) de alta sofisticação com gradientes decorativos sutis.
   - Tipografia refinada utilizando a fonte moderna de alta legibilidade **Inter** combinada a elementos técnicos em **JetBrains Mono**.
   
2. **Simulador Tributário Interativo**:
   - Diagnóstico em tempo real que calcula a alíquota tributária anual estimada e o imposto pago, sugerindo se há cenários de economia otimizada (Simples Nacional, Lucro Presumido, etc.).
   - Teaser direcionando a empresa para uma consulta com dados simulados pré-preenchidos.

3. **Fluxo de Atendimento Sem Barreiras**:
   - Remoção de formulários de e-mail longos e estressantes.
   - Integração de um **botão gigante premium de contato imediato via WhatsApp** com redirecionamento de mensagem contextualizada.
   - Badge flutuante interativo de suporte humanizado ativo.

4. **Painel de metodologia (Assessoria)**:
   - Estrutura clara e dividida em 3 passos para remoção do receio sobre a migração de contabilidade: Diagnóstico Inicial, Transição Transparente e Atendimento Ativo (<15 minutos de resposta).

5. **Localização Integrada (Santos/SP)**:
   - Seção de contato com mapa nativo Google Maps de Vila Matias integrado de forma translúcida com badge de compromisso híbrido.

---

## 🛠️ Tecnologias Utilizadas

- **React 18** com **Vite** como empacotador ultrarrápido.
- **Tailwind CSS** para estilização utilitária responsiva premium e efeitos de blur backdrop.
- **TypeScript** para garantia total de tipagem e integridade do código.
- **Lucide React** para um conjunto moderno e homogêneo de ícones limpos.
- **Motion/React (Framer Motion)** para animações e micro-interações suaves de entrada e scroll.

## 📁 Estrutura de Pastas

```bash
/src
  ├── components/          # Componentes modulares reutilizáveis
  │     ├── Assessoria.tsx      # Bloco metodológico de mudança sem dor
  │     ├── Contact.tsx         # Contatos corporativos, mapa e botão gigante Zap
  │     ├── FeatureCounter.tsx  # Métricas e selos de credibilidade social proof
  │     ├── Footer.tsx          # Rodapé regulatório (CNPJ, endereço)
  │     ├── Hero.tsx            # Dobra de entrada de alto apelo estratégico
  │     ├── Navbar.tsx          # Cabeçalho fixo com navegação suave
  │     ├── Services.tsx        # Catálogo de especialidades focadas
  │     └── Simulator.tsx       # Simulador de economia de impostos
  ├── types.ts             # Dados centrais unificados da empresa (Telefones, CNPJ)
  ├── App.tsx              # Componente principal unificador do frame geral
  └── main.tsx             # Arquivo de entrada do React
```

## ⚡ Como Rodar o Projeto

1. Instale as dependências:
   ```bash
   npm install
   ```
2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
3. Construa o projeto para produção:
   ```bash
   npm run build
   ```
